# Особливості інтерпретації даних

Оброблені дані, а саме значення ЗВО, розміщуються у відкритому доступі на сайті **WOUDC**, британської антарктичної служби з деякою затримкою, яка виникає в необхідності постобробки рядів даних з метою:

- фільтрації викидів
- усунення технічних помилок виявлених за допомогою візуального контролю
